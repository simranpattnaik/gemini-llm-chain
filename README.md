# LangChain Gemini 

This project is a simple **LLM-based question answering system** built using **LangChain and Google Gemini**.

It takes user input as a question and generates a response using a prompt template and Gemini model. The goal of this project is to understand how to connect LLMs with LangChain and create a basic AI workflow.

---

## Project Overview

In this project, I created a system where:
- user enters a question
- the question is passed into a prompt template
- the template is sent to Gemini model
- the model generates a response

This helps in understanding how LLM pipelines work step by step.
